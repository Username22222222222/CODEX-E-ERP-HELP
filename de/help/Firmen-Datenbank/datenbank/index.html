    Einordnung

    X-ERP Hilfe > Firmen-Datenbank > Datenbank

    Jede Firma in X-ERP besitzt ihre eigene Datenbank, in der alle Stamm- und Bewegungsdaten gespeichert werden. Sie liegt auf einem Microsoft SQL Server und bildet die zentrale, gesicherte Datenbasis Ihres Mandanten. Aufbau und Pflege der Datenbank sind Grundlage für Datensicherheit, Auswertbarkeit und einen stabilen Betrieb.
