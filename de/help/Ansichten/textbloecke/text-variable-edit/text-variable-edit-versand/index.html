    TextVariableEdit-Versand: Versand in X-ERP

    Das Register Versand in TextVariableEdit ordnet die Felder, Listen und Aktionen dieses Bereichs fachlich ein.

    Bereich: HilfeKeyword: ERPEbene: 3Original: ShippingTechnischer Name: TextVariableEdit-Versand

    Einordnung

    Ansichten > Textblöcke > TextVariableEdit > TextVariableEdit-Versand

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Textnummer

      Textnummer ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      IdOriginal: Eindeutige Nummer.

    2. Name

      Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      NameOriginal: Bezeichnung des Datensatzes.
