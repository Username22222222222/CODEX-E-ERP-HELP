    PackingListBoxEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in PackingListBoxEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: PackingListBoxEdit-Übersicht

    Einordnung

    Ansichten > Packen > PackingListBoxEdit > PackingListBoxEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. ID

      ID ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      IdOriginal: Eindeutige interne Kennung des Datensatzes.

    2. Kommissionierliste

      Das Feld Kommissionierliste im Register Kommissionierliste in PackingListBoxEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      PickingListDocIdOriginal: Eingabefeld für: Kommissionierliste.

      DisplayMatchcodeOriginal: Kurzbezeichnung zum schnellen Suchen und Finden.
