    PackingListBoxPositionEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in PackingListBoxPositionEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: PackingListBoxPositionEdit-Übersicht

    Einordnung

    Ansichten > Packen > PackingListBoxPositionEdit > PackingListBoxPositionEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Box ID

      Box ID ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      BoxIdOriginal: Eingabefeld für: Box ID.

    2. Boxtyp

      Boxtyp ordnet den Datensatz fachlich ein. Diese Zuordnung erleichtert Auswertung, Steuerung und strukturierte Bearbeitung.

      DisplayPackingListBoxTypeOriginal: Eingabefeld für: Boxtyp.

    3. Kommissionierliste

      Das Feld Kommissionierliste im Register Kommissionierliste in PackingListBoxPositionEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      DisplayDocNumberOriginal: Eingabefeld für: Kommissionierliste.

    4. Datum der Kommissionierliste

      Legt das relevante Datum oder den Zeitpunkt für 'Datum der Kommissionierliste' fest. Diese Angabe wird für Planung, Filterung und Nachvollziehbarkeit verwendet.

      DisplayPickingListDocDateOriginal: Relevantes Datum.

    5. Versandnummer

      Versandnummer ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      DisplayShippingNumberOriginal: Eindeutige Nummer.

    6. Position Nr.

      Das Feld Position Nr. im Register Position Nr. in PackingListBoxPositionEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      DisplayPositionNumberOriginal: Eingabefeld für: Position Nr..

    7. Dok Position

      Das Feld Dok Position im Register Dok Position in PackingListBoxPositionEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      DocPositionIdOriginal: Eingabefeld für: Dok Position.

    8. Artikelnummer

      Artikelnummer ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      DisplayArticleNumberOriginal: Eindeutige Artikelnummer.

    9. Artikel-Matchcode

      Artikel-Matchcode beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      DisplayArticleMatchcodeOriginal: Codeschlüssel zur Klassifizierung.

    10. Menge

      Enthält die Mengenangabe 'Menge'. Sie wird für Verfügbarkeit, Planung, Lagerung oder Disposition verwendet.

      QuantityOriginal: Mengenangabe.

    11. Serie / Charge

      Das Feld Serie / Charge im Register Serie / Charge in PackingListBoxPositionEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      IndividualizationNumberIdOriginal: Eingabefeld für: Serie / Charge.

    12. Matchcode

      Kurzbezeichnung zum schnellen Suchen und Finden.

      DisplayPartnerMatchcodeOriginal: Kurzbezeichnung zum schnellen Suchen und Finden.

    13. Info

      Info enthält ergänzende Informationen für Dokumentation, Druckausgabe oder interne Abstimmung.

      InfoOriginal: Zusätzliche Informationen zum Datensatz.
