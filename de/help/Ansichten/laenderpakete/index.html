    Länderpakete: ERP-Ansichten im Überblick

    Dieses Unterverzeichnis bündelt die EditViews und Wizards im Bereich Länderpakete.

    Bereich: HelpPageKeyword: ERPEbene: 1Original: CountryPackageTechnischer Name: Länderpakete

    Einordnung

    Ansichten > Länderpakete

    Unterseiten

    CountryPackageDatevAccountNumberWizard: Land Package Datev Konto Number Assistent Der CountryPackageDatevAccountNumber-Assistent führt Anwender Schritt für Schritt durch den zugehörigen Prozess im Bereich Länderpakete. Die Hilfe erklärt Zweck, Eingaben und Aktionen so, dass der Ablauf sicher nachvollzogen werden kann. CountryPackageDatevExportEdit: Land Package Datev Export bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Datev Export. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Kontenplan, Datev-Mandant, Beraternummer und Beratername. CountryPackageElsterExportEdit: Land Package Elster Export bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Elster Export. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Name, Rechtsform, Straße und PLZ. CountryPackageIntrastatExportEdit: Land Package Intrastat Export bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Intrastat Export. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Datum, Typ, Buchungsperiode und Meldeland. CountryPackageIntrastatExportTransactionEdit: Land Package Intrastat Export Transaction bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Intrastat Export Transaction. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Report, Dok Position, Warencode und Datum. CountryPackagePrivacyProcessingActivityEdit: Land Package Privacy Processing Activity bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Privacy Processing Activity. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Name, Zweck, Rechtsgrundlage und Kategorien betroffener Personen. CountryPackagePrivacyProcessingActivityObjectEdit: Land Package Privacy Processing Activity Object bearbeiten Diese Ansicht pflegt landesspezifische Vorgaben für Country Package Privacy Processing Activity Object. Sie ist wichtig für gesetzliche Meldungen, Formate, Datenschutz und regionale ERP-Prozesse. Wichtige Angaben sind Objekttyp, Schema, Objektname und Feldname.
