    ExtraApiEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in ExtraApiEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: ExtraApiEdit-Übersicht

    Einordnung

    Ansichten > Extra-Tabellen > ExtraApiEdit > ExtraApiEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Name

      Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      NameOriginal: Bezeichnung des Datensatzes.

    2. Info

      Info enthält ergänzende Informationen für Dokumentation, Druckausgabe oder interne Abstimmung.

      InfoOriginal: Zusätzliche Informationen zum Datensatz.
