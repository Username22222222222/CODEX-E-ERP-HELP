    ExtraCodeEdit: Extra Code bearbeiten

    Diese Ansicht pflegt Extra Code, mit denen X-ERP ohne Standardanpassung erweitert werden kann. Dadurch lassen sich zusätzliche Daten, Tabellen und Funktionen kontrolliert in Prozesse einbinden. Wichtige Angaben sind Code-ID (nur A-Z und 0-9, 'XX' wird automatisch vorangestellt), Name und Code.

    Bereich: ViewKeyword: ERPEbene: 2Technischer Name: ExtraCodeEdit

    Einordnung

    Ansichten > Extra-Tabellen > ExtraCodeEdit

    Unterseiten
