    RoleEdit-Berechtigungen: Berechtigungen in X-ERP

    Das Register Berechtigungen in RoleEdit ordnet die Felder, Listen und Aktionen dieses Bereichs fachlich ein.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: RoleEdit-Berechtigungen

    Einordnung

    Ansichten > Benutzer > RoleEdit > RoleEdit-Berechtigungen

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Rollenname

      Bezeichnet die Rolle, die Benutzern zugewiesen wird.

      RollennameOriginal: Eingabefeld für: Rollenname.

    2. Seitenname

      Nennt die X-ERP-Seite, für die Berechtigungen gesetzt werden.

      SeitennameOriginal: Eingabefeld für: Seitenname.

    3. WebApi Name

      Nennt die technische API-Funktion, für die Berechtigungen gesetzt werden.

      WebApi NameOriginal: Eingabefeld für: WebApi Name.

    4. Lesen

      Erlaubt das Anzeigen von Daten oder Funktionen.

      LesenOriginal: Eingabefeld für: Lesen.

    5. Erstellen

      Erlaubt das Anlegen neuer Datensätze.

      ErstellenOriginal: Eingabefeld für: Erstellen.

    6. Ändern

      Erlaubt das Bearbeiten vorhandener Datensätze.

      ÄndernOriginal: Eingabefeld f\u00fcr: Ändern.

    7. Löschen

      Erlaubt das Entfernen von Datensätzen oder Rechten.

      LöschenOriginal: Eingabefeld f\u00fcr: Löschen.

    8. Alle auswählen

      Setzt die sichtbaren Berechtigungen gesammelt. Diese Aktion sollte bewusst verwendet werden.

      Alle auswählenOriginal: Schaltfl\u00e4che: Alle auswählen.

    9. Alle entfernen

      Entfernt die sichtbaren Berechtigungen gesammelt. Diese Aktion ist für Rollenbereinigung und Tests hilfreich.

      Alle entfernenOriginal: Schaltfläche: Alle entfernen.
