    Benutzer: ERP-Ansichten im Überblick

    Der Bereich Benutzer bündelt die zugehörigen ERP-Ansichten in X-ERP. Anwender finden hier Bearbeitungsmasken, Register, Felder und Aktionen in fachlicher Reihenfolge.

    Bereich: HelpPageKeyword: ERPEbene: 1Original: UserLoginSecurityTechnischer Name: Benutzer

    Einordnung

    Ansichten > Benutzer

    Unterseiten

    UserEdit: Benutzer bearbeiten UserEdit verwaltet ERP-Benutzer, Anmeldung, Sprache, Rolle, Passwortregeln und die Verknüpfung zum Mitarbeiter. Die Ansicht ist sicherheitsrelevant, weil sie festlegt, wer sich anmelden darf, welche Rolle verwendet wird und ob ein Konto aktiv, gesperrt oder zur Passwortänderung verpflichtet ist. User Culture Setting UserCultureSetting ist die Bearbeitungsansicht fuer User Culture Setting im Bereich Benutzer. Die Hilfe erklaert Zweck, Register, Felder und Aktionen, damit Anwender die Maske sicher ausfuellen und den Vorgang richtig einordnen koennen. User Login Setting UserLoginSetting ist die Bearbeitungsansicht fuer User Login Setting im Bereich Benutzer. Die Hilfe erklaert Zweck, Register, Felder und Aktionen, damit Anwender die Maske sicher ausfuellen und den Vorgang richtig einordnen koennen. RoleEdit: Role bearbeiten RoleEdit verwaltet Berechtigungen und Systemmenüs einer Benutzerrolle. Administratoren legen je Seite und WebApi fest, ob Lesen, Erstellen, Ändern oder Löschen erlaubt ist. Dadurch wird X-ERP auf Aufgabenbereiche zugeschnitten und sensible ERP-Funktionen werden kontrolliert freigegeben. AuthorizationPageEdit: Authorization Page bearbeiten AuthorizationPageEdit ist die Bearbeitungsansicht fuer Authorization Page. Sie dient im Bereich Benutzer dazu, Autorisierung, Seitenrechte und WebApi-Rechte. Anwender pruefen und pflegen hier insbesondere Id. AuthorizationWebApiEdit: Authorization Web Api bearbeiten AuthorizationWebApiEdit ist die Bearbeitungsansicht fuer Authorization WebApi. Sie dient im Bereich Benutzer dazu, Autorisierung, Seitenrechte und WebApi-Rechte. Anwender pruefen und pflegen hier insbesondere Id.
