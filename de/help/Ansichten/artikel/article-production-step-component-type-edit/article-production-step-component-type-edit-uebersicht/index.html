    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Name

      Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      NameOriginal: Bezeichnung des Datensatzes.

    2. Lagerungswirksam

      Enthält eine lagerbezogene Angabe für 'Lagerungswirksam'. Sie wird für Bestand, Verfügbarkeit und logistische Prozesse verwendet.

      WarehousingEffectiveOriginal: Lagerbezogene Angabe.

    3. Umsatzwirksam

      Das Feld Umsatzwirksam im Register Umsatzwirksam in ArticleProductionStepComponentTypeEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      RevenueEffectiveOriginal: Eingabefeld für: Umsatzwirksam.
