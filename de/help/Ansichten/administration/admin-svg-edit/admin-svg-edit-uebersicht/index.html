    AdminSvgEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in AdminSvgEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: AdminSvgEdit-Übersicht

    Einordnung

    Ansichten > Administration > AdminSvgEdit > AdminSvgEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. SVG-Name

      SVG-Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      IdOriginal: Bezeichnung bzw. Name.

    2. SVG-Code

      SVG-Code ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      SvgCodeOriginal: Codeschlüssel zur Klassifizierung.
