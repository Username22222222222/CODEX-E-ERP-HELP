    AdminProgramSettingEdit: Administration Program Einstellung bearbeiten

    Diese Ansicht bündelt Einstellungen, die das Verhalten von X-ERP im Bereich Administration steuern. Sie ist wichtig, weil hier Vorgaben gepflegt werden, die sich auf viele nachgelagerte Prozesse auswirken. Wichtige Angaben sind Längeneinheit, Gewichtseinheit, Volumeneinheit und Zeiteinheit.

    Bereich: ViewKeyword: ERPEbene: 2Technischer Name: AdminProgramSettingEdit

    Einordnung

    Ansichten > Administration > AdminProgramSettingEdit
