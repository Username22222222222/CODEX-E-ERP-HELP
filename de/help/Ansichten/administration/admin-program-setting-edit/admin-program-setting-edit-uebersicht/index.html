    AdminProgramSettingEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in AdminProgramSettingEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: AdminProgramSettingEdit-Übersicht

    Einordnung

    Ansichten > Administration > AdminProgramSettingEdit > AdminProgramSettingEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Längeneinheit

      Das Feld Längeneinheit im Register Längeneinheit in AdminProgramSettingEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      UnitOfLengthOriginal: Eingabefeld für: Längeneinheit.
