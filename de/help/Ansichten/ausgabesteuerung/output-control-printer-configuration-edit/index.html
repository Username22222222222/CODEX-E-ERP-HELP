    OutputControlPrinterConfigurationEdit: Output Control Printer Configuration bearbeiten

    Diese Ansicht verwaltet Drucker- und Ausgabekonfigurationen. Sie steuert, welche Drucker, Mengen und Ausgabewege für Belege oder interne Dokumente verwendet werden. Wichtige Angaben sind Name, Drucker und Menge.
