    ReportTemplateEdit-Benutzer und Rollen: Benutzer und Rollen in X-ERP

    Das Register Benutzer und Rollen in ReportTemplateEdit ordnet die Felder, Listen und Aktionen dieses Bereichs fachlich ein.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: ReportTemplateEdit-Benutzer und Rollen

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Report Name

      Bezeichnet die Reportvorlage eindeutig für Auswahl, Druckvorschau und Ausgabesteuerung.

      Report NameOriginal: Eingabefeld für: Report Name.

    2. Report Template XML

      Enthält die technische Layoutdefinition der Vorlage. Änderungen sollten nur durch berechtigte Anwender erfolgen.

      Report Template XMLOriginal: Eingabefeld für: Report Template XML.

    3. Report Type

      Ordnet die Vorlage einer Reportart zu. Diese Zuordnung steuert, wo die Vorlage angeboten wird.

      Report TypeOriginal: Eingabefeld für: Report Type.

    4. Dok-Typ

      Verknüpft die Vorlage mit einem Dokumenttyp, damit Belege die passende Ausgabe verwenden.

      Dok-TypOriginal: Eingabefeld für: Dok-Typ.
