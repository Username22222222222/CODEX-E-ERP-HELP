    ResourceEdit-Details: Details in X-ERP

    Das Register Details in ResourceEdit enthält ergänzende Fachinformationen, die für Auswertungen, Druck, Shop-Anbindung oder weitere Prozesse benötigt werden.

    Bereich: HilfeKeyword: ERPEbene: 3Original: DetailsTechnischer Name: ResourceEdit-Details

    Einordnung

    Ansichten > Ressourcen > ResourceEdit > ResourceEdit-Details

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Hersteller

      Das Feld Hersteller im Register Details in ResourceEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      ManufacturerIdOriginal: Hersteller des Artikels.

    2. Hersteller-Bestellnummer

      Hersteller-Bestellnummer ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      ManufacturerOrderNumberOriginal: Eindeutige Nummer.
