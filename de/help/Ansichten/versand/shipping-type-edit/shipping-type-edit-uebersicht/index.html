    ShippingTypeEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in ShippingTypeEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: ShippingTypeEdit-Übersicht

    Einordnung

    Ansichten > Versand > ShippingTypeEdit > ShippingTypeEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Name

      Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      NameOriginal: Bezeichnung des Datensatzes.

    2. Versandartencode

      Versandartencode ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      ShippingTypeCodeOriginal: Codeschlüssel zur Klassifizierung.

    3. Carrier

      Das Feld Carrier im Register Carrier in ShippingTypeEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      CarrierCodeOriginal: Eingabefeld für: Carrier.

    4. Url

      Das Feld Url im Register Url in ShippingTypeEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      UrlOriginal: Internetadresse (URL).
