    FinanceTransferCategoryEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in FinanceTransferCategoryEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: FinanceTransferCategoryEdit-Übersicht

    Einordnung

    Ansichten > Finanzwesen > FinanceTransferCategoryEdit > FinanceTransferCategoryEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Name

      Name beschreibt den Datensatz so, dass Anwender ihn in Listen, Suchfeldern und Auswertungen schnell wiedererkennen.

      NameOriginal: Bezeichnung des Datensatzes.

    2. Betreff

      Das Feld Betreff im Register Betreff in FinanceTransferCategoryEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      SubjectOriginal: Kurze Betreffzeile zum Inhalt.

    3. Kontentyp

      Kontentyp ordnet den Datensatz fachlich ein. Diese Zuordnung erleichtert Auswertung, Steuerung und strukturierte Bearbeitung.

      AccountTypeIdOriginal: Eingabefeld für: Kontentyp.

    4. Sachkonto

      Enthaelt eine kaufmaennische Angabe in FinanceTransferCategoryEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      GeneralLedgerAccountIdOriginal: Zugeordnetes Konto.

    5. Steuerschlüssel

      Enthaelt eine kaufmaennische Angabe in FinanceTransferCategoryEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      TaxKeyIdOriginal: Schlüssel zur Steuerermittlung.

    6. Personenkonto

      Enthaelt eine kaufmaennische Angabe in FinanceTransferCategoryEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      PartnerDetailIdOriginal: Zugeordnetes Konto.
