    FinanceAccountPurposeEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in FinanceAccountPurposeEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. ID

      ID ist eine Kennung für Suche, Zuordnung oder externe Referenzierung. Sie hilft, Datensätze eindeutig wiederzufinden.

      IdOriginal: Eindeutige interne Kennung des Datensatzes.

    2. Kontozweck

      Enthaelt eine kaufmaennische Angabe in FinanceAccountPurposeEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      NameOriginal: Zugeordnetes Konto.
