    FinanceTaxKeyDetailEdit-Übersicht: Übersicht in X-ERP

    Das Register Übersicht in FinanceTaxKeyDetailEdit fasst die wichtigsten Stammdaten zusammen und dient als erster Kontrollpunkt für Suche, Identifikation und schnelle Bearbeitung.

    Bereich: HilfeKeyword: ERPEbene: 3Technischer Name: FinanceTaxKeyDetailEdit-Übersicht

    Einordnung

    Ansichten > Finanzwesen > FinanceTaxKeyDetailEdit > FinanceTaxKeyDetailEdit-Übersicht

    Felder und Elemente

    Diese Einträge erscheinen direkt in diesem Bereich der Ansicht.

    1. Steuerschlüssel

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      TaxKeyIdOriginal: Schlüssel zur Steuerermittlung.

    2. Gültig ab

      Das Feld Gültig ab im Register Gültig ab in FinanceTaxKeyDetailEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      ValidFromOriginal: Eingabefeld für: Gültig ab.

    3. Steuer %

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      TaxPercentOriginal: Steuerrelevante Angabe.

    4. Erwerbsteuerkonto

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      TaxAccountIdOriginal: Zugeordnetes Konto.

    5. Steuerkonto / Erwerbsteuerkonto

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      TaxAccountIdOriginal: Zugeordnetes Konto.

    6. Skontokonto

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      CashDiscountAccountIdOriginal: Zugeordnetes Konto.

    7. Nicht-Fällig-Konto / Vorsteuerkonto

      Das Register Nicht-Fällig-Konto / Vorsteuerkonto fasst die zugehoerigen Angaben der Ansicht FinanceTaxKeyDetailEdit zusammen. Hier stehen die Felder und Aktionen, die fuer diesen Teil des Vorgangs gemeinsam bearbeitet oder geprueft werden.

      NotDueAccountIdOriginal: Zugeordnetes Konto.

    8. Vorsteuerkonto

      Enthaelt eine kaufmaennische Angabe in FinanceTaxKeyDetailEdit. Sie wirkt auf Bewertung, Buchhaltung, Preisfindung oder finanzielle Auswertungen.

      NotDueAccountIdOriginal: Zugeordnetes Konto.

    9. Kurs

      Das Feld Kurs im Register Kurs in FinanceTaxKeyDetailEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      RateOriginal: Wechselkurs der Währung.

    10. Exportschlüssel

      Das Feld Exportschlüssel im Register Exportschlüssel in FinanceTaxKeyDetailEdit speichert eine fachliche Information, die fuer Pflege, Suche, Auswertung oder Weiterverarbeitung dieses ERP-Vorgangs relevant ist.

      ExportKeyOriginal: Eingabefeld für: Exportschlüssel.
