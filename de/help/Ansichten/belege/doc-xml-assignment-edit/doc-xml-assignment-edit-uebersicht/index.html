      Enthaelt eine technische XML-Definition. Aenderungen sollten nur durch berechtigte Anwender mit Kenntnis der jeweiligen Ausgabe- oder Schnittstellenstruktur erfolgen.

      Das Register Dok-Typ fasst die zugehoerigen Angaben der Ansicht DocXmlAssignmentEdit zusammen. Hier stehen die Felder und Aktionen, die fuer diesen Teil des Vorgangs gemeinsam bearbeitet oder geprueft werden.
