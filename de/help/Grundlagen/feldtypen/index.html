    Diese Abschnitte gehören fachlich zu diesem Bereich und folgen der Struktur aus der Excel-Hilfe.

    Textfeld Das Textfeld ist das Standard-Eingabefeld für kurze, einzeilige Texte wie Bezeichnungen, Namen oder Codes. Es nimmt Buchstaben, Zahlen und Sonderzeichen... Mehrzeiliges Textfeld Das mehrzeilige Textfeld dient zur Erfassung längerer Texte wie Bemerkungen oder Beschreibungen. Mit der Eingabetaste können Zeilenumbrüche gesetzt... Code Feld (ohne Leerzeichen) Dieses Textfeld verhindert die Eingabe von Leerzeichen und eignet sich für Schlüssel, Kürzel oder Codes, die zusammenhängend geschrieben werden müssen... E-Mail Feld Das E-Mail-Feld nimmt eine E-Mail-Adresse entgegen und prüft das Format. Aus dem Feld heraus kann häufig direkt eine E-Mail erstellt werden. (Komponente... Telefonfeld Das Telefonfeld dient der Erfassung von Telefon-, Mobil- oder Faxnummern und prüft die zulässigen Zeichen. (Komponente XPhon) Link Feld Das Link-Feld speichert eine Web-Adresse (URL). Der hinterlegte Link lässt sich anklicken und öffnet die Zielseite im Browser. (Komponente XLink) Ganzzahlfeld Das Ganzzahlfeld akzeptiert nur ganze Zahlen, zum Beispiel Stückzahlen, Indizes oder Anzahlen. Buchstaben werden nicht angenommen. (Komponente XInte) Dezimalfeld Das Dezimalfeld dient zur Eingabe von Zahlen mit Nachkommastellen. Die Anzahl der angezeigten Nachkommastellen richtet sich nach der jeweiligen... Mengenfeld Das Mengenfeld erfasst Mengenangaben, meist in Verbindung mit einer Mengeneinheit. (Komponente XQuan) Gewichtsfeld Das Gewichtsfeld dient zur Eingabe von Gewichten und formatiert den Wert entsprechend. (Komponente XWeig) Längenfeld Das Längenfeld erfasst Längen- bzw. Maßangaben als Zahl mit Nachkommastellen. (Komponente XLeng) Prozentfeld Das Prozentfeld nimmt einen Prozentwert entgegen und stellt ihn mit Prozentzeichen dar, zum Beispiel Rabatte oder Wahrscheinlichkeiten. (Komponente XPerc) Datumsfeld Das Datumsfeld dient zur Eingabe eines Datums. über ein Kalendersymbol lässt sich das Datum bequem auswählen; je nach Feld ist zusätzlich eine... Zeitfeld Das Zeitfeld dient zur Eingabe einer Uhrzeit (Stunden/Minuten). (Komponente XTime) Kontrollkästchen Das Kontrollkästchen (Checkbox) stellt einen Ja/Nein- bzw. An/Aus-Wert dar. Ein Haken bedeutet "ja"/"aktiv", ein leeres Kästchen "nein"/"inaktiv"... Optionsfeld Das Optionsfeld (Radio-Button) bietet mehrere sich gegenseitig ausschließende Auswahlmöglichkeiten an; es kann immer nur eine Option aktiv sein... Auswahlfeld (Dropdown) Das Auswahlfeld zeigt beim Anklicken eine Aufklappliste mit vordefinierten Werten, aus denen genau einer gewählt wird. (Komponente XSele) Nachschlagefeld (Lookup) Das Nachschlagefeld (Lookup) verknüpft den aktuellen Datensatz mit einem anderen (z. B. Partner, Artikel). Beim Tippen werden passende Einträge... Sprachfeld Das Sprachfeld dient zur Auswahl einer Sprache bzw. Länderkultur und beeinflusst z. B. Formate und übersetzungen.
