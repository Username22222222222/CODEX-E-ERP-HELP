    Artikel Set

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Stammdaten > Artikel > Artikel Set

    Ein Artikel-Set ist eine Art Bündel- oder Paket-Artikel: Ein einziger Artikel besteht aus mehreren anderen Artikeln (den Komponenten). Beispiel: Der Set-Artikel "Geschenkkorb" setzt sich aus den Komponenten "Flasche Wein", "Glas" und "Verpackung" zusammen.

    In diesem Tab legen Sie fest, aus welchen Komponenten das Set besteht. Pro Komponente geben Sie an:

    • Komponente: welcher Artikel Teil des Sets ist.
    • Mengenfaktor: wie oft diese Komponente im Set enthalten ist (z. B. 6 Flaschen).
    • Kostenlos: die Komponente wird mitgeliefert, aber nicht berechnet (Gratis-Beigabe).
    • Auflösen: beim Verkauf wird das Set in seine Einzelkomponenten aufgelöst und diese werden statt des Sets auf dem Beleg angezeigt.
    • Komponente drucken: legt fest, ob diese Komponente auf Belegen (z. B. Lieferschein, Rechnung) gedruckt wird.

    Der Tab erscheint nur, wenn der Artikel als Set gekennzeichnet ist und bereits gespeichert wurde.

    Unterseiten

    Diese Abschnitte gehören fachlich zu diesem Bereich und folgen der Struktur aus der Excel-Hilfe.

    Komponente Artikel, der als Komponente in diesem Set enthalten ist. Mengenfaktor Menge der Komponente, die pro Set-Einheit benötigt wird. Kostenlos Kennzeichnet die Komponente als kostenlos (kein Einzelpreis im Set). Auflösen Steuert, ob das Set im Beleg in seine Einzelkomponenten aufgelöst dargestellt wird. Komponente drucken Legt fest, ob die Komponente auf dem Beleg gedruckt wird.
