    Zielfeld

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Customizing > Formatierte Suche > Zielfeld

    Das Zielfeld nimmt das Ergebnis der formatierten Suche auf und wird automatisch befüllt. Problem: Ein ermittelter Wert nützt nur, wenn er auch an der richtigen Stelle landet. Hier bestimmen Sie genau, welches Feld das Ergebnis erhält. So reduzieren Sie manuelle Eingaben gezielt und sorgen dafür, dass abhängige Felder stets konsistente, plausible Werte enthalten.
