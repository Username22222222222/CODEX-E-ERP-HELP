    Erwähnung

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Module > Intercom > Erwähnung

    Über eine Erwähnung sprechen Sie gezielt eine Kollegin oder einen Kollegen an, die oder der daraufhin benachrichtigt wird. So stellen Sie sicher, dass wichtige Hinweise die richtige Person erreichen. So bleibt nichts liegen, und Zuständigkeiten sind unmittelbar klar.
