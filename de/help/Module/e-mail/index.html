    E-Mail

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Module > E-Mail

    Das Modul **E-Mail** integriert den gesamten E-Mail-Verkehr direkt in X-ERP. Belege wie Angebote, Auftragsbestätigungen, Lieferscheine oder Rechnungen können ohne Medienbruch direkt aus dem ERP-System versendet werden.

    Ein- und ausgehende E-Mails werden den zugehörigen Partnern zugeordnet. Dadurch entsteht eine vollständige und nachvollziehbare Kommunikationshistorie.

    Gleichzeitig behalten Sie die volle Datenhoheit über Ihre Kommunikation: Alle E-Mails werden auf Ihrem eigenen Server gespeichert. So bleiben Sie unabhängig von cloudbasierten E-Mail-Providern und behalten jederzeit die Kontrolle über Ihre geschäftliche Korrespondenz.

    Zur Erhöhung der Sicherheit können E-Mails von unbekannten Absenderadressen automatisch in separate Ordner verschoben werden. Dadurch lassen sich Phishing-Angriffe und unerwünschte Nachrichten wirksam reduzieren.
