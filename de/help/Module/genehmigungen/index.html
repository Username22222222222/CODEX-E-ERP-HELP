    Einordnung

    X-ERP Hilfe > Module > Genehmigungen

    Das Modul **Genehmigungen** unterstützt die Umsetzung von Freigabeprozessen für Belege und Vorgänge.

    Genehmigungsworkflows können flexibel gestaltet werden und unterstützen das **Vier- oder Mehr-Augen-Prinzip**.

    So schaffen Sie transparente Entscheidungswege, erhöhen die Prozesssicherheit.
