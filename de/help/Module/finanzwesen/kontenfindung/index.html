    Kontenfindung

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Module > Finanzwesen > Kontenfindung

    Die Kontenfindung steuert, auf welche Sachkonten Geschäftsvorfälle automatisch gebucht werden – abhängig von Artikel, Partner, Steuer und Vorgangsart. Einmal eingerichtete Regeln sorgen dafür, dass jeder Beleg korrekt verbucht wird, ohne dass Anwender Konten manuell wählen müssen. Das beschleunigt die Buchhaltung und vermeidet Fehlbuchungen.
