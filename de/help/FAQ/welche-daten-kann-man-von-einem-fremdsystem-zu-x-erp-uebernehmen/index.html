    Welche Daten kann man von einem Fremdsystem zu X-ERP übernehmen?

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > FAQ > Welche Daten kann man von einem Fremdsystem zu X-ERP übernehmen?

    Die meisten Stammdaten können direkt übernommen und anschließend in X-ERP weitergenutzt werden. Dazu gehören beispielsweise Artikel, Partner, Kunden, Lieferanten, Preise und weitere zentrale ERP-Daten. Bewegungsdaten wie Rechnungen, Gutschriften oder historische Vorgänge können als Archivdaten in Extra-Tabellen übernommen werden. Für Migrationen aus bekannten Fremdsystemen stehen vorbereitete Übernahme- und Archivskripte bereit, sodass die Datenübernahme weitgehend automatisiert erfolgen kann.
