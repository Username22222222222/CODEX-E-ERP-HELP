    Was mache ich im Fall einer Störung?

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > FAQ > Was mache ich im Fall einer Störung?

    Gehen Sie bei einer anhaltenden Fehlermeldung schrittweise vor:

    1. Schließen Sie alle Browserfenster und starten Sie X-ERP oder die betroffene Portal-App neu.
    2. Besteht das Problem weiterhin, starten Sie den Client neu.
    3. Tritt auch danach keine Besserung ein, starten Sie den Server neu.
    4. Bleibt der Fehler unverändert bestehen, übermitteln Sie Ihrem Fachhandelspartner den exakten Wortlaut der Fehlermeldung und die Schritte, mit denen das Problem reproduziert werden kann.
