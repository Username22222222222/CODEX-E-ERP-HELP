    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server

    Der Server ist das Herzstück der Installation: Hier laufen Datenbank und Anwendung, auf die alle Arbeitsplätze zugreifen. Dieser Abschnitt erläutert Voraussetzungen, Vorbereitung und Einrichtung des Servers samt SQL Server und Webserver (IIS). Eine saubere Servereinrichtung ist die Grundlage für Leistung, Sicherheit und Stabilität.
