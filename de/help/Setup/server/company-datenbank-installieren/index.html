    ERP Hilfe

    Company Datenbank installieren

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > Company Datenbank installieren

    Mit dem Company-Creator legen Sie nun die eigentliche Firmendatenbank auf dem SQL Server an. Sie nimmt alle Stamm- und Bewegungsdaten Ihres Mandanten auf. Nach Anlage steht die Datenbank bereit, um aktiviert und mit Daten gefüllt zu werden.
