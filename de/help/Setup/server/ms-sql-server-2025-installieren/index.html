    ERP Hilfe

    MS SQL Server 2025 installieren

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > MS SQL Server 2025 installieren

    In diesem Schritt installieren Sie Microsoft SQL Server 2025 als Datenbankserver für X-ERP. Sie führen das Setup aus, legen eine Instanz an und wählen die grundlegenden Optionen. Eine korrekte SQL-Server-Installation ist die Voraussetzung dafür, dass die Firmendatenbanken angelegt und betrieben werden können.
