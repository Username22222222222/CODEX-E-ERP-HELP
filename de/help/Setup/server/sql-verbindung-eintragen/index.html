    SQL Verbindung eintragen

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > SQL Verbindung eintragen

    Hier hinterlegen Sie die Verbindung zum SQL Server – Servername, Instanz und Anmeldedaten. über diese Verbindung greift X-ERP auf die Datenbanken zu. Eine korrekte Eingabe ist Voraussetzung dafür, dass die Anwendung das XLD und die Firmendatenbanken erreichen kann.
