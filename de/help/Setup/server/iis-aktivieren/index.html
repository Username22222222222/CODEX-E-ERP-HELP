    ERP Hilfe

    IIS aktivieren

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > IIS aktivieren

    X-ERP wird über die Internet Information Services (IIS) als Webanwendung bereitgestellt. IIS ist ein Windows-Feature, das vor der Installation aktiviert werden muss. Dieser Schritt richtet den Webserver ein, über den die Arbeitsplätze später auf X-ERP zugreifen.
