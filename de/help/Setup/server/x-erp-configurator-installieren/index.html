    ERP Hilfe

    X ERP Configurator installieren

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > X ERP Configurator installieren

    Der X-ERP-Configurator ist das Werkzeug, mit dem Sie die Installation konfigurieren und verwalten – etwa SQL-Verbindung, Ordner, Lizenz und Zertifikat. In diesem Schritt installieren Sie den Configurator. Er begleitet Sie anschließend durch die eigentliche Konfiguration des Systems.
