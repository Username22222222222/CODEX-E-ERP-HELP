    X ERP Configurator öffnen

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > X ERP Configurator öffnen

    Über den X-ERP-Configurator verwalten Sie die Installation auch nach der Ersteinrichtung – etwa um Einstellungen anzupassen, Zertifikate zuzuweisen oder Bindings zu pflegen. Dieser Schritt zeigt, wie Sie den Configurator öffnen. Er bleibt das zentrale Werkzeug für Konfiguration und Wartung.
