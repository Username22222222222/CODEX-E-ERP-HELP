    Einordnung

    X-ERP Hilfe > Setup > Server > .NET Hosting Bundle installieren

    Das .NET Hosting Bundle stellt die Laufzeitumgebung bereit, die X-ERP im IIS benötigt. Es enthält die ASP.NET-Core-Runtime und das IIS-Modul und muss nach der Aktivierung des IIS installiert werden. Erst danach kann der Webserver die X-ERP-Anwendung korrekt ausführen.
