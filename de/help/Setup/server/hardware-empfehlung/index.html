    ERP Hilfe

    Hardware Empfehlung

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Server > Hardware Empfehlung

    Wählen Sie eine angemessene Ausstattung, die genügend Reserven für Wachstum bietet. Das stellt einen flüssigen Betrieb sicher und schützt vor späteren Engpässen.

    Mindestkonfiguration: Betriebssystem: Windows Server 2025 (für Testzwecke kann auch Windows 11 genutzt werden) CPU: 8 Kerne Intel Xeon Silver min. 3 GHz Arbeitsspeicher: 32 GB RAM Systemlaufwerk: 500 GB NVMe SSD, 7000 MB/s Lesen, 5000 MB/s Schreiben Datenlaufwerk: 1 TB NVMe SSD (separat für SQL-Datenbanken empfohlen), 7000 MB/s Lesen, 5000 MB/s Schreiben Netzwerk: 1 Gbit/s Backup: Tägliche SQL-Backups auf separates Speichermedium
