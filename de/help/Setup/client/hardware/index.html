    Hardware

    Bereich: HilfeKeyword: ERP

    Einordnung

    X-ERP Hilfe > Setup > Client > Hardware

    Für den Client genügt handelsübliche Hardware, da die eigentliche Verarbeitung auf dem Server stattfindet. Wichtig sind ein aktueller Browser und eine ausreichend schnelle Netzwerkanbindung. So können auch ältere Geräte komfortabel mit X-ERP arbeiten, was Investitionen in Arbeitsplatzhardware gering hält.
